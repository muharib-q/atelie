# АтельеШик — Веб-приложение на Python (Flask)

## Структура проекта
```
atelier/
├── app.py              ← главный файл (сервер Flask)
├── database.db         ← база данных SQLite (создаётся автоматически)
├── requirements.txt    ← зависимости
├── templates/
│   ├── index.html      ← главная страница (Jinja2-шаблон)
│   └── admin.html      ← панель администратора
└── static/
    ├── css/style.css   ← стили
    ├── js/main.js      ← JavaScript
    └── img/            ← изображения
```

## Как запустить

### 1. Установить Python (если не установлен)
Скачать с https://python.org/downloads (версия 3.8+)

### 2. Установить Flask
Открыть командную строку (cmd) в папке проекта:
```
pip install flask
```

### 3. Запустить сервер
```
python app.py
```

### 4. Открыть сайт
В браузере перейти на: http://localhost:5000

### 5. Панель администратора (просмотр заявок)
http://localhost:5000/admin

## Как работает приложение
- При запуске `app.py` создаётся база данных `database.db` (SQLite)
- Все данные (услуги, мастера, отзывы) хранятся в БД
- Форма записи отправляет данные на сервер (POST /booking)
- Сервер сохраняет заявку в таблицу `bookings`
- Администратор видит все заявки на странице /admin
