ТРЕБОВАНИЯ:
  - Python 3.8 или новее (скачать: python.org)
  - Интернет для установки Flask (один раз)

ШАГ 1 — Установить Flask (один раз):
  Открыть командную строку (cmd) и написать:
  pip install flask

ШАГ 2 — Запустить сервер:
  Перейти в папку с проектом:
  cd путь/к/папке/atelier

  Запустить:
  python app.py

ШАГ 3 — Открыть сайт:
  Браузер откроется автоматически, или вручную:
  Сайт:  http://127.0.0.1:5000
  Админ: http://127.0.0.1:5000/admin

СТРУКТУРА ПРОЕКТА:
  app.py              — главный файл (сервер Python/Flask)
  database.db         — база данных SQLite (создаётся автоматически)
  templates/
    index.html        — шаблон главной страницы (Jinja2)
    admin.html        — панель администратора
  static/
    css/style.css     — стили сайта
    js/main.js        — JavaScript (анимации, маска телефона)
    img/              — фотографии (about, gallery)

ТЕХНОЛОГИИ:
  - Python 3 + Flask — серверная часть
  - SQLite           — база данных
  - Jinja2           — шаблонизатор HTML
  - HTML5/CSS3/JS    — клиентская часть

ПАНЕЛЬ АДМИНИСТРАТОРА (/admin):
  - Просмотр всех заявок от клиентов
  - Изменение статуса: новая / в работе / выполнена / отменена
  - Удаление заявок
